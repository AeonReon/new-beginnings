The Feast

A 52-week rotation. Each week one poem, one hymn, one composer piece, one painting, one story, one nature prompt, one practical-life activity. You are not expected to do all of it. Open the app, pick what fits today.

Start date

The app calculates the current week from the day you first opened the Feast. Change the start date below if you'd like to reset or align it with a real calendar.

On "completing" a feast

The completion checkboxes are for you, not a scorecard. A week where you only read the poem and went outside is a good week. A week where you checked everything off but felt hurried is not.

Checking things serves attention (you notice what you did and did not do) but harms presence if it becomes the point. The rule of thumb: if you're checking to feel better, stop. If you're checking to remember, continue.
